# living-pusher

### 推流助手小程序

+ 粘贴、扫码快速输入推流地址 (支持 flv, rtmp 格式) 只支持微信小程序
+ 三种分辨率选择
+ 竖屏、横屏直播
+ 支持预览
+ 查看推流日志、美颜、切换前后摄像头、静音、闪光灯
+ 首次直播前需要同意麦克风和摄像头授权

```
yarn  // 安装依赖
yarn dev:weapp  // 开发
yarn build:weapp  // 打包
```
